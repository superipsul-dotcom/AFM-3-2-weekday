# 🍚 돼지고기 옥수수 솥밥

고소한 옥수수와 부드러운 돼지고기가 어우러진 한 그릇 솥밥. 쫀득한 누룽지와 짭조름한 양념장으로 마무리하면 별다른 반찬 없이도 완성되는 든든한 한 끼입니다.

---

## 📋 기본 정보

| 항목 | 내용 |
| --- | --- |
| 분류 | 한식 · 밥 요리 |
| 인분 | 2인분 |
| 조리 시간 | 약 40분 |
| 난이도 | ★★☆☆☆ (보통) |

## 🛒 재료

### 밥 재료
- 쌀 2컵 (불린 쌀 기준)
- 물 1¾컵 (솥밥은 평소보다 약간 적게)
- 옥수수 알갱이 1컵 (생옥수수·통조림·냉동 모두 가능)
- 돼지고기 (앞다리살 또는 목살) 200g
- 당근 ¼개 (다지기)
- 표고버섯 2개 (선택, 채썰기)

### 돼지고기 밑간
- 간장 1큰술
- 맛술(청주) 1큰술
- 다진 마늘 ½큰술
- 후추 약간
- 참기름 ½큰술

### 양념장
- 간장 3큰술
- 물 1큰술
- 다진 파 2큰술
- 고춧가루 ½큰술
- 깨소금 ½큰술
- 참기름 1큰술
- 다진 마늘 ½큰술

## 👩‍🍳 조리 순서

1. **쌀 불리기** — 쌀을 깨끗이 씻어 30분 정도 물에 불린 후 체에 받쳐 물기를 뺍니다.
2. **돼지고기 밑간** — 한입 크기로 썬 돼지고기에 밑간 재료를 넣고 10분간 재웁니다.
3. **재료 손질** — 옥수수는 물기를 빼고, 당근과 표고버섯은 잘게 썰어 준비합니다.
4. **고기 볶기** — 달군 솥(또는 냄비)에 참기름을 두르고 밑간한 돼지고기를 겉면이 익을 정도로 볶습니다.
5. **쌀 안치기** — 볶은 고기 위에 불린 쌀을 평평하게 올리고, 물 1¾컵을 부은 뒤 옥수수·당근·표고버섯을 골고루 얹습니다.
6. **밥 짓기** — 센 불로 끓이다가 끓어오르면 약불로 줄여 12~13분, 불을 끄고 10분간 뜸을 들입니다.
7. **양념장 만들기** — 뜸 들이는 동안 양념장 재료를 모두 섞어 둡니다.
8. **마무리** — 밥을 골고루 섞은 후 그릇에 담고, 기호에 따라 양념장을 곁들여 비벼 드세요.

---

## 💡 요리 팁

- 🌽 **옥수수**: 생옥수수를 쓰면 단맛이 가장 좋습니다. 통조림은 물기를 충분히 빼주세요.
- 🔥 **누룽지**: 뜸을 들인 뒤 1~2분 더 센 불로 가열하면 바닥에 고소한 누룽지가 생깁니다.
- 🥩 **돼지고기**: 기름이 적당히 있는 목살·앞다리살이 부드럽고 풍미가 좋습니다.
- 🧈 **고소함 강화**: 밥이 다 되면 버터 한 조각을 올려 섞으면 풍미가 깊어집니다.
- ⏱ **솥밥 물 양**: 일반 밥보다 물을 약 10% 적게 잡아야 질지 않고 고슬고슬합니다.
